# 功能簡介

### 簡易登入頁面。
### 使用者輸入帳號密碼，若為已註冊成員則顯示登入畫面，否則回到登入頁並顯示錯誤訊息。


# 開啟流程

**※請事先安裝Node.js**

1. Clone該專案至本地
2. 以npm安裝所需套件(詳見下方使用工具)
3. 在終端機上輸入
`node app.js`
4. 如出現「Succeed in listening on http://localhost:3000」表示啟動成功，請進入該網址
